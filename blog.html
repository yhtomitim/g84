Learning to Learn firebase

I learned how to deploy websites using Firebase. I was familiar with using Wordpress and GitHub pages, but not Firebase. I think this skill will help in quickly deploying an app to the web 2) allowing for quick "in the wild" testing.

As part of the learn to learn process, I am working on "clearing the stage." The video below shows my actual process for clearnig the stage on my Mac, but I put my iPhone in Do Not Disturb mode, and I listen to some classical music to block out distraction. When I forget to clear the stage, I find myself answering all my notifications, making it very hard to stay on track.

To make sure I'm learning well, I really enjoy the "play" compoenent of learn to learn. With firebase, I really enjoyed seeing how my changes were uploaded almost instantly with firebase deploy. Also, I had enabled 2-factor authentication on my google account, so it was neat to look up how to create a two-factor app password. I enjoyed that we can use it from the terminal.

The reflection component of learn to learn is important to commit what I learned to long-term memory. I hope that by using firebase, I can very quickly deploy sites and troubleshoot. I also think that using firebase is a good employable skill.
